Ride Analytics

A react app built to analyze a given data set, particularly ride data for a cab service or so. Can infer ride frequency with respect to a variety of factors such as time, mode , ride length and monthly frequency. Built with react, redux for the UI, react-mapbox-gl for the map, extra style provided by the mapbox-gl library. Graphs made with react-vis library by Uber.
Requires a very particular data set and will not work with any generic data, yet.
